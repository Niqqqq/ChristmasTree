# 3D动态圣诞树 - 照片墙效果

一个使用Python实现的3D动态圣诞树，具有照片墙的散开聚合动画效果。

## 功能特点

- 🎄 3D圣诞树形状，由多个照片粒子组成
- 📸 每个粒子都是随机颜色的"照片"
- ✨ 散开聚合动画效果
- 🔄 自动旋转视角
- 🎮 交互式控制

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python main.py
```

## 控制说明

- **空格键**: 切换散开/聚合模式
- **A键**: 开启/关闭自动动画
- **方向键**: 手动旋转视角
  - 左/右: 水平旋转
  - 上/下: 垂直旋转

## 效果说明

程序会创建一个由照片粒子组成的3D圣诞树：
- 粒子会自动在散开和聚合状态之间切换
- 每个粒子都有旋转动画
- 相机视角会自动旋转，展示3D效果
- 粒子按深度排序，实现正确的3D遮挡效果

## 技术实现

- 使用Pygame进行图形渲染
- 使用NumPy进行数学计算
- 使用PIL生成随机颜色的照片粒子
- 3D到2D的透视投影
- 平滑的动画插值
